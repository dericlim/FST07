Initialising git repository
- git init

Add to staging area
- git add .

Make it another version
- git commit -m "[Message Description]"

Show history of commits
- git log

Show history of commits of one line only
- git log --oneline

Legend in git repository; changes in colours and below words will be displayed
U - Untracked
A - Added
M - Modified

Check on the working tree / status of our commit
- git status

Creating a branch
- git branch [branchname]

List all the branches
- git branch -a

Switch to another branch
- git checkout [branchname]

Merging master to branch
- git merge [branchname]

Deleting a branch
- git branch -d [branchname]
